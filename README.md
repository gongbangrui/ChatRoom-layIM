# 欢迎使用 ChatRoom-LayIm

------

本聊天室是基于Typecho+LayIm开发的聊天室插件

[LayIm](http://layim.layui.com/)

[代码地址](https://github.com/gaobinzhan/ChatRoom-layIM)

部署方法：

下载插件包 放在 `/usr/plugins/`下 修改目录名为ChatRoom

[在线预览](https://blog.gaobinzhan.com/)
